I've added an "Apps" page for small webapps. Check out the (small) list of webapps. Who knows, you might find one of them useful. I plan to add a few more apps later, but it's nice to get the architecture in place with a few already launched. There are currently two apps: str2md5 converts a given string to MD5, and Steam ID and Community ID is a useful tool for finding IDs that Steam uses if you have one of the other. Great for server admins who want to ban previous players found in the playerlist.
